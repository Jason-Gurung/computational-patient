# Demo Ideas — Hardcoded Frontend Concepts

Ideas for visualizing and demoing the vision. These would be hardcoded/mocked to show the concept, not functional backends.

## Pfizer Brain Demo Screens

### 1. The Conversational Interface
- Teams-like chat interface where a user asks natural language questions
- Hardcoded responses showing how the system answers differently based on user persona
- Same question, side-by-side: developer view vs. marketer view vs. scientist view

### 2. Project Knowledge Graph
- Interactive visualization of projects as nodes, relationships as edges
- Click a project to see its AI-generated summary, team, status, related projects
- Visual indicators for: active / stale / shelved / orphaned
- Animated connections showing when the system detects duplication or overlap

### 3. User Profile & Proactive Recommendations
- Mock user dashboard showing: "Here's what's relevant to you today"
- Personalized feed based on role, active projects, and recent queries
- Recommendations: tools to try, people to connect with, relevant shelved projects

### 4. Staleness Detection Dashboard
- Projects on a health spectrum (green → yellow → red)
- Timeline view showing activity decay
- The graduated response flow: automated check-in → escalation → human investigation

## Computational Patient Demo Screens

### 5. The Trial Pipeline Comparison
- Side-by-side: traditional timeline vs. computational-layer timeline
- Animated flow showing how the model feeds into and learns from each phase
- Cost/time counters ticking down as the model catches problems earlier

### 6. Virtual Patient Population
- Visualization of thousands of simulated patients as a particle cloud
- Color-coded by predicted response (responder / non-responder / adverse reaction)
- Interactive: adjust the drug parameters and watch the population response shift

### 7. Phase III Decision Support
- Mock dashboard showing model predictions for a Phase III go/no-go decision
- Probability gauges, subgroup breakdowns, alternative trial designs
- "What if" scenarios: what if we enrich for biomarker X? What if we reduce sample size?

### 8. The Compounding Model
- Timeline showing model accuracy improving across successive drug programs
- Drug 1 → Drug 10 → Drug 50 accuracy curves
- Visualization of the data moat growing over time

## Information Medium Demo Screens

### 9. "Talk to a Project"
- Interface where you can ask a project questions directly
- Shows how it pulls from code, commits, conversations, Jira — not a static doc
- Side-by-side: the old way (reading a stale Confluence page) vs. the new way

### 10. Decision Archaeology
- Timeline of decisions for a project, automatically extracted from conversations and code
- Click any decision to see: who made it, what alternatives were considered, what the consequences were
- No human authored any of this — it was all comprehended from existing artifacts
